# Kids Puzzle Game

A simple and educational puzzle game for children aged 4-10.

## Goal

Upload any image, choose puzzle size, and solve the puzzle by arranging shuffled pieces.

## Core Features

- Upload custom image
- Grid sizes:
  - 2x2
  - 3x3
  - 4x4
  - 5x5
  - 6x6
- Shuffle puzzle pieces
- Drag and drop support
- Touch support for tablets
- Hint system
- Ghost image mode
- Victory detection
- Score tracking

## Target Users

Children aged 4-10.

## Design Principles

- Large buttons
- Large puzzle pieces
- Bright colors
- Minimal text
- Touch-friendly UI

## Tech Stack

- React
- TypeScript
- Vite
- dnd-kit

## Non Goals

Do NOT implement:

- Backend
- User accounts
- Authentication
- Multiplayer
- Microservices
- AI features
- Database

Keep the project simple.
